# TaskSphere CI/CD Project
## Docker Compose

Configured Docker Compose to orchestrate the React frontend and Django backend services.

### Features

- Multi-container application
- Backend service
- Frontend service
- Shared Docker network
- Simplified deployment using a single command
## GitHub Actions CI

Configured a GitHub Actions Continuous Integration (CI) workflow to automatically validate the TaskSphere application.

### Features

- Automatic workflow execution on push and pull request
- Django dependency installation
- React dependency installation
- React production build
- Django system checks
- Docker image build validation
